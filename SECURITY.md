# Security Policy

Sanjie 是开源 demo / 产品原型，不承诺生产级安全能力。请不要把真实个人信息、账号密码或敏感业务数据写入演示库。

## 支持范围

当前仅维护主分支最新代码。历史版本不单独发布安全修复。

## 报告方式

如果发现安全问题，请通过 GitHub issue 提供最小复现信息，避免公开真实密钥、数据库或个人信息。

报告建议包含：

- 受影响模块或接口。
- 复现步骤。
- 影响范围。
- 建议修复方向。

## 当前限制

- 登录是演示身份切换，不是生产认证。
- `X-User-ID` 是 mock auth，不应用于真实权限场景。
- SQLite 默认适合本地 demo 和单机部署。
- Docker Compose 配置面向本地部署示例，不包含 TLS、WAF、备份和监控。
