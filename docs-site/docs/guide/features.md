# 功能介绍

Sanjie 的产品风格是“严肃移动办公 + 荒诞神话业务”。页面和流程按真实管理系统设计，文案和数据保持玩笑性质。

## 工作台

工作台采用“三界总控大屏”风格，聚合当前角色最关心的信息：今日勾魂任务、待审批数量、投胎积压、地狱高负载楼层、孟婆汤库存预警、生死簿高危变更和未处理告警。

工作台指标、三界流转图、天条预警和推荐演示路线都可以直接跳转到对应业务页。核心列表页和详情页不套同一模板，而是按业务分别设计成夜巡任务、奈何桥票据、判官案牍、命簿卷宗、地狱控制台和许愿签文，适合在文档站内嵌 H5 演示中直接走查。

推荐演示路径：工作台查看风险指标，进入勾魂或审批详情完成一次状态流转，再进入生死簿、刑期详情或愿望详情触发冻结、复核、转派等操作，最后到审计日志确认留痕。

## 勾魂管理

勾魂任务面向黑白无常和鬼差：

- 查看待执行、执行中、异常、已完成任务，列表页采用夜巡雷达任务池。
- 查看目标魂魄、地点、计划时间和执行人。
- 详情页采用夜巡定位风格，展示任务摘要、执行闭环、异常说明和下一步操作提示。
- 支持开始勾魂、完成任务、上报异常、申请撤销。
- 异常类型包括目标抢救中、勾错魂、魂魄逃逸、查无此人、疑似飞升。

## 投胎管理

投胎队列管理魂魄进入轮回前的流程：

- 查看魂魄编号、目标道别、优先级、孟婆汤状态和胎位资源，列表页采用奈何桥排队票。
- 详情页采用奈何桥排队票据风格，展示轮回档案、审核流程、胎位/汤剂状态和下一步建议。
- 支持通过、驳回、转复核。
- 支持分配富贵胎、普通胎、清贫胎、天才胎和困难模式胎。
- 支持孟婆汤发放和遗忘状态确认。

## 审批中心

审批中心承接所有高危流程：

- 寿命变更审批。
- 投胎插队审批。
- 勾魂异常审批。
- 地狱刑期复核。
- 神仙下凡备案。

审批列表页采用案卷堆样式，突出案由、目标卷宗、申请人和审批人。

审批详情页采用判官案牍风格，展示审批档案、申请理由、处理结果、审批路径和风险盖章。审批动作包括同意、驳回和转交，所有操作进入审计日志。

## 生死簿

生死簿提供移动端轻量查询：

- 按姓名、生辰信息、风险等级搜索。
- 列表页采用命簿索引册样式，直接展示阳寿刻度和封印状态。
- 详情页采用命簿卷宗风格，展示阳寿偏差、命格、劫数、死因、风险等级和锁定策略。
- 冻结高危记录，并把风险等级提升为严重。
- 发起寿命变更申请，生成审批单而不是直接改写阳寿。
- 查看相关审计日志，适合演示“改寿必须留痕”的闭环。

## 十八层地狱

十八层地狱用于查看楼层运行状态：

- 楼层容量、当前在押数、负载等级。
- 设备状态和告警。
- 楼层与刑期列表采用地狱控制台样式，突出负载条、设备状态和刑期仪表。
- 刑期详情采用地狱控制台风格，展示魂魄、楼层、罪行、刑期、痛感和执行设备。
- 刑期复核申请会进入审批中心，并写入审计日志。
- 高负载楼层跨层分流。

## 愿望工单

愿望工单处理凡人的许愿请求：

- 求财、求姻缘、求考试、求升职等类型。
- 根据香火、功德、合理性判断优先级。
- 列表页采用许愿票据样式，展示香火/功德权重和承办神仙。
- 详情页采用许愿签文风格，展示许愿人、香火、功德、承办神仙、处理建议和系统吐槽。
- 支持转派给财神、月老、文昌帝君、太白金星等。
- 支持办结、驳回和结果备注，mock 演示会即时刷新状态。

## 孟婆汤

孟婆汤模块管理库存和发放：

- 汤坊库存台展示标准版、低糖版、加浓版等库存。
- 库存低于预警线时提醒。
- 支持在移动端输入调整数量和备注，完成补货或扣减。
- 发放后记录剂量和记忆残留。
- 禁止库存调整为负数。

## 审计日志

审计日志记录寿命、投胎、地狱刑期、勾魂异常、愿望处理和汤坊库存等敏感操作。页面采用天眼留痕时间线，可按审批、命簿、勾魂、地狱、愿望和汤坊过滤，主要用于防止批量划名、关系户投胎和审批路径异常。

## 角色身份

角色身份用于演示不同岗位的移动端工作方式：

- 我的页展示仙籍令牌、当前接口模式、角色推荐入口和权限模拟。
- 切换身份页提供玉帝、阎王、判官、黑无常、孟婆、天兵统领、城隍和巡天审计使。
- mock 演示会把当前身份写入本地存储，后续操作会按该身份写入审计记录。
