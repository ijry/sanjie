# 功能介绍

Sanjie 的产品风格是“严肃移动办公 + 荒诞神话业务”。页面和流程按真实管理系统设计，文案和数据保持玩笑性质。

## 工作台

工作台聚合当前角色最关心的信息：今日勾魂任务、待审批数量、投胎积压、地狱高负载楼层、孟婆汤库存预警和生死簿高危变更。

工作台指标卡和快捷入口都可以直接跳转到对应业务页。生死簿、十八层地狱、愿望工单、孟婆汤和审计日志均按移动端高密度卡片展示，适合在文档站内嵌 H5 演示中直接走查。

## 勾魂管理

勾魂任务面向黑白无常和鬼差：

- 查看待执行、执行中、异常、已完成任务。
- 查看目标魂魄、地点、计划时间和执行人。
- 详情页展示任务摘要、执行闭环、异常说明和下一步操作提示。
- 支持开始勾魂、完成任务、上报异常、申请撤销。
- 异常类型包括目标抢救中、勾错魂、魂魄逃逸、查无此人、疑似飞升。

## 投胎管理

投胎队列管理魂魄进入轮回前的流程：

- 查看魂魄编号、目标道别、优先级、孟婆汤状态和胎位资源。
- 详情页展示轮回档案、审核流程、胎位/汤剂状态和下一步建议。
- 支持通过、驳回、转复核。
- 支持分配富贵胎、普通胎、清贫胎、天才胎和困难模式胎。
- 支持孟婆汤发放和遗忘状态确认。

## 审批中心

审批中心承接所有高危流程：

- 寿命变更审批。
- 投胎插队审批。
- 勾魂异常审批。
- 地狱刑期复核。
- 神仙下凡备案。

审批详情页展示审批档案、申请理由、处理结果、审批路径和风险提示。审批动作包括同意、驳回和转交，所有操作进入审计日志。

## 生死簿

生死簿提供移动端轻量查询：

- 按姓名、生辰信息、风险等级搜索。
- 查看阳寿、命格、劫数、死因和锁定状态。
- 冻结高危记录。
- 发起寿命变更申请。
- 查看相关审计日志。

## 十八层地狱

十八层地狱用于查看楼层运行状态：

- 楼层容量、当前在押数、负载等级。
- 设备状态和告警。
- 刑期记录和罪行类型。
- 刑期复核申请。
- 高负载楼层跨层分流。

## 愿望工单

愿望工单处理凡人的许愿请求：

- 求财、求姻缘、求考试、求升职等类型。
- 根据香火、功德、合理性判断优先级。
- 支持转派给财神、月老、文昌帝君等。
- 支持办结、驳回和结果备注。

## 孟婆汤

孟婆汤模块管理库存和发放：

- 查看标准版、加浓版等库存。
- 库存低于预警线时提醒。
- 发放后记录剂量和记忆残留。
- 禁止库存调整为负数。

## 审计日志

审计日志记录寿命、投胎、地狱刑期、勾魂异常等敏感操作。它是系统里最严肃的部分，主要用于防止批量划名、关系户投胎和审批路径异常。
